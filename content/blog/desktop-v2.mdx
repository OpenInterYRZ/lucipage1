## Why a Rewrite

After a year of iteration, we learned what works and what doesn't. Desktop 2.0 is built from the ground up with those lessons.

## Performance

The numbers speak for themselves:

- **3x faster** memory indexing
- **60% less** memory usage
- **Instant** search results, even with millions of memories
- **50% smaller** app size

### Native Performance

We moved critical paths to native code, giving you the performance of a native app with the flexibility of web technologies.

## Redesigned Interface

The new interface is cleaner, faster, and more intuitive:

- **Command palette** — access any feature with a keyboard shortcut
- **Split view** — work with multiple contexts side by side
- **Timeline view** — browse your history visually
- **Focus mode** — minimize distractions while LUCI works in the background

## New Features

- **Quick capture** — save thoughts and screenshots with a global shortcut
- **Smart folders** — auto-organized collections based on your projects
- **Widget system** — customize your dashboard with the info you need

## Upgrade

Desktop 2.0 is a free update for all existing users. Your data migrates automatically.
